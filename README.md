Local Development Setup

1. Clone the repository:
   git clone <REPO_GIT_URL>

2. Open the project folder:
   cd <PROJECT_NAME>

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

Editing the Project
You can edit the project locally using any IDE like VS Code or directly from GitHub by opening the file, editing and committing changes.

Deployment (Cloudflare Pages)

This project will be deployed using Cloudflare Pages.

Steps:

1. Push your code to GitHub

2. Go to Cloudflare Pages

3. Create a new Project

4. Connect your GitHub repository

5. Select the project

6. Set Build Command:
   npm run build

7. Set Output Folder:
   dist

8. Deploy the project

Cloudflare will automatically build and host the website.

Custom Domain (Cloudflare)

1. Open Cloudflare Pages Project
2. Go to Custom Domains
3. Add your domain
4. Update DNS if required
5. Save and verify
